Limp Wrist

Just where have we been anyways?

Download this project as a .zip file Download this project as a tar.gz file

I'm creating a database and map of of all the concerts Limp Wrist has ever played since our first show in 1999. This map is hosted on Cartodb and synched to my Google database, and will eventually contain a photo in each info-window headers. More info about Limp Wrist is available on our bandcamp site.
